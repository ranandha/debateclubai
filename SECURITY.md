# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please **do not** open a public issue. Email the maintainers at:

- `security@debateclubai.example`

Include a clear description, reproduction steps, and any relevant logs or screenshots. We will respond within 7 days.

## Scope

This repo is a **public demo application** intended for local development and private use. It stores API keys in the browser (IndexedDB) and uses optional client-side encryption.

## Safe Usage Guidelines

- Do **not** deploy this app publicly without authentication and a secure server-side storage solution.
- Avoid using the app on shared or untrusted machines.
- Rotate keys if you suspect exposure.

## Supported Versions

Only the latest `main` branch is supported for security fixes.
